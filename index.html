HTML

  • The head element contains information (metadata) about the webpage.
  • The body element represents the visible content shown to the user.
  • The DOCTYPE (which is not case sensitive) declaration tells the browser what document type it should expect when it is opened.
  • The lang="en" element tells the browser the language for the webpage (English in this example).
  • The title element defines the title for a webpage, which is !important! for SEO (Search Engine Optimization,the process of improving the ranking of a website in search engines).
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport. information about the page that is used by the browser. The user wonâ€™t see information in the "head" or "meta" elements.
  • The script element embeds client-side scripts in the HTML document.
  • Elements listed inside other elements are called child elements, which are contained by parent elements.
  • Use Semantic Elements when possible. Semantic elements define their meaning such as form or table, whereas div and span do not.
